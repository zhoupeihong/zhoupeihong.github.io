CSS (Cascading Style Sheets)

Language for stylizing HTML files

  • Selector

    • Points to the HTML element you want to style

  • Property

    • Stylistic feature you want to change

    • Common CSS Properties

      • font-size

        • Used to size the text of a tag

      • line-height

        • Used to size the height of text lines

      • color

        • Used to set the text colour of an element

  • Value

    • How you want to change its stylistic feature
