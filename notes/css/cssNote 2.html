Notes

ID

  • text-align

    • Specifies the horizontal alignment of text

    • left

      • Aligns the text to the left

    • right

      • Aligns the text to the right

    • center

      • Centers the text

  • text-decoration

    • Adds an underline, over-line, line-through, to selected text

    • none

      • No text-decoration

    • underline

      • Horizontal line underneath text

    • overline

      • Horizontal line above text

    • line-through

      • Strikethrough. Horizontal line through the middle of text

  • font-weight

    • Bold or light text

    • normal

      • Standard weight. Equivalent of 400

    • bold

      • Bold weight. Equivalent of 700

    • 100

      • Lightest

    • 900

      • Boldest

  • font-family

    • Specifies the font for an element

    • serif

      • Small line attached to the end of a stroke in a letter

    • sans-serif

      • Does not have extending features at the end of strokes

    • monospace

      • Fixed-width font (Code font)

    • cursive

      • Flowing font

    • fantasy

      • Daft font

  • <span>

    • Used for grouping and applying styles to inline elements

  • id

    • Specifies a unique id for an HTML element
    Can be used on any HTML element
    When using an ID, the selector should start with `#`
