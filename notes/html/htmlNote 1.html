html Elements

  • <html>

    • The first element in HTML
    The entire content of the HTML file is inside it

    • <head>

      • Where you put information that isn't visible in the website

      • <title>

        • Title that is displayed on the top of the page

    • <body>

      • Main content of the page, containing text, images, tables, etc.

      • <p> (Paragraph)

        • Creates a paragraph

      • <h1> (Header)

        • Used as the header of the document
        The size decreases as the numbers change from 1 to 6

      • <ul> (Unordered List)

        • Creates an empty unordered list

      • <li> (List Item)

        • Creates one bullet

      • <ol> (Ordered List)

        • Creates an empty ordered list
