Semantic Elements

  • <header>

    • A container for introductory content
    A <header> element typically contains:

    • One or more heading elements (<h1> - <h6>)
    • Logo or icon

  • <nav>

    • A container for a set of navigation links

  • <section>

    • Defines sections in a document

  • <footer>

    • Container for the bottom of the webpage
